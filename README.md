# 🌦️ OpenWeather App

A simple weather application built using **Kotlin**, **MVVM architecture**, and **Jetpack Compose**.
This app fetches real-time weather data using the OpenWeather API and also includes **(sign in and registration)** powered by **Firebase Firestore**.

---

## ✨ Features

* 🌍 Get current weather by city
* 📋 View weather list/forecast
* ⚡ Built with MVVM architecture
* 🔄 State management using StateFlow
* 🎨 Modern UI with Jetpack Compose

---

## 📦 Tech Stack

* Kotlin
* Jetpack Compose
* MVVM (Model-View-ViewModel)
* StateFlow
* Retrofit (for API calls)

---

## 🚀 Getting Started

### 1. Clone the repository

```bash
git clone https://github.com/lukiesluke88/openweather.git
```
---

### 2. Setup API Key 🔐

To run this project, you need to provide your own weather API key.

#### Steps:

1. Create a new file named `secrets.properties` in the **root directory** of the project.

2. Add your API key:

```properties
WEATHER_API_KEY=YOUR_API_KEY
```

3. Replace `YOUR_API_KEY` with your actual API key.
